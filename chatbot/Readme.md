Usage
1.Enter Your Message: Type your query or message into the text box on the left.

2.Submit: Press the "Submit" button to send your message to the chatbot.

3.Chat with the Bot: The bot will reply based on predefined responses.

4.Conversation History: The conversation between you and the chatbot will be displayed on the screen.

Customizing Intents :

1.The chatbot uses an intents.json file that stores different patterns (keywords) and their corresponding responses.

2.You can modify the intents.json file to add new intents, patterns, and responses according to your requirements.

Sample intents.json format:
Structure:
1.patterns: A list of input patterns (phrases) that the chatbot can recognize.

2.responses: A list of responses the bot will give when it matches any of the patterns.
